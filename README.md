# 🛒 Интернет-магазин - Фильтрация и сортировка товаров

Учебный проект интернет-магазина на Django с реализацией фильтрации, сортировки, пагинации и асинхронной подгрузкой данных через AJAX.

## 📋 Функционал

### Основные возможности:

- **Просмотр каталога товаров** с фильтрацией по:
  - Категории товара
  - Диапазону цен
  - Дате добавления (последние 7/30/90 дней)
- **Сортировка товаров** по:
  - Популярности (количество заказов)
  - Цене (возрастание/убывание)
  - Дате добавления (новые/старые сначала)
- **Пагинация** - 10 товаров на странице
- **AJAX подгрузка** - фильтрация и сортировка без перезагрузки страницы
- **Дополнительные страницы**:
  - Популярные товары (топ-20)
  - Недавние заказы (пользователи с заказами за 30 дней)


## 📝 Администрирование

### Создание заказов через админку

1. Перейдите в `/admin/`
2. Создайте заказ в разделе **Заказы**
3. Через **Товары в заказе** добавьте позиции (товары + количество)

## 🗄️ Модели базы данных


```
### Product (Товар)
| Поле | Тип | Описание |
|------|-----|----------|
| name | CharField | Название товара |
| category | CharField | Категория товара |
| price | DecimalField | Цена товара |
| date_added | DateTimeField | Дата добавления (авто) |

### Order (Заказ)
| Поле | Тип | Описание |
|------|-----|----------|
| user | ForeignKey(User) | Пользователь |
| created_at | DateTimeField | Дата создания |

### OrderItem (Товар в заказе)
| Поле | Тип | Описание |
|------|-----|----------|
| order | ForeignKey(Order) | Заказ |
| product | ForeignKey(Product) | Товар |
| quantity | IntegerField | Количество |
```


## 🌐 Доступные страницы:


* `http://127.0.0.1:8000/` → редирект на каталог
* `/shop/` → Каталог с фильтрацией (главная)
* `/shop/popular/` → Популярные товары
* `/shop/recent-orders/` → Недавние заказы
* `/admin/` → Админ-панель


## 🎯 Используемые технологии

* **Backend** : Django 4.x
* **Frontend** : HTML5, CSS3, Vanilla JavaScript
* **Асинхронность** : Fetch API (AJAX)
* **База данных** : SQLite
* **Стили** : Custom CSS (Flexbox, Grid, адаптивная вёрстка)
