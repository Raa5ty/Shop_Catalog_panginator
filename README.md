# 🛒 Интернет-магазин - Фильтрация и сортировка товаров

## 📋 Функционал

#### Основные возможности:

- **Просмотр каталога товаров** с фильтрацией по:
- Категории товара
- Диапазону цен
- Дате добавления
- **Сортировка товаров** по популярности, цене и дате
- **Пагинация** - 10 товаров на страницу
- **AJAX подгрузка** - фильтрация и сортировка без перезагрузки

#### Расширенный CRUD:

- ✅ Создание, редактирование, удаление товаров
- ✅ Создание, редактирование, удаление категорий
- ✅ Валидация цены (не может быть ниже 0)
- ✅ Подтверждение перед удалением

#### Аналитика:

- 📊 Общая стоимость всех товаров
- 📊 Количество товаров в каждой категории
- 📊 Средняя, минимальная и максимальная цена по категориям
- 📊 Общая статистика каталога

#### Дополнительно:

- 🖼️ Загрузка изображений для товаров (Pillow)
- 🎨 Интерфейс на Bootstrap 5
- 📱 Адаптивная вёрстка

#### Создание заказов через админку(временно)

1. Перейдите в `/admin/`
2. Создайте заказ в разделе **Заказы**
3. Через **Товары в заказе** добавьте позиции (товары + количество)

## 🗄️ Модели базы данных

```
### Category (Категория)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| name | CharField(100) | Название категории (уникальное) |
| description | TextField | Описание категории |
| created_at | DateTimeField | Дата создания (авто) |

### Product (Товар)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| name | CharField(100) | Название товара |
| category | ForeignKey(Category) | Категория (связь) |
| price | DecimalField | Цена (валидация ≥ 0) |
| description | TextField | Описание товара |
| image | ImageField | Изображение товара |
| date_added | DateTimeField | Дата добавления (авто) |
| updated_at | DateTimeField | Дата обновления (авто) — добавлено в new версии |

### Order (Заказ)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| user | ForeignKey(User) | Пользователь |
| created_at | DateTimeField | Дата создания |

### OrderItem (Товар в заказе)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| order | ForeignKey(Order) | Заказ |
| product | ForeignKey(Product) | Товар |
| quantity | IntegerField | Количество |

### Связи между моделями:

Category ──┬── Product ──┬── OrderItem ──┬── Order ──── User
│ │ │
└── has many ──┘ └── has many ──┘
```

## 🌐 URL-маршруты:

* URL	Страница	Описание
* /	→ Редирект	Автоматический переход в каталог
* /shop/	Каталог	Главная страница с фильтрацией (AJAX)
* /shop/products/	Список товаров	Альтернативный список
* /shop/product/`<id>`/	Детали товара	Просмотр товара
* /shop/product/create/	Создание	Новый товар
* /shop/product/`<id>`/edit/	Редактирование	Изменение товара
* /shop/product/`<id>`/delete/	Удаление	Подтверждение удаления
* /shop/categories/	Категории	Список категорий
* /shop/category/create/	Создание	Новая категория
* /shop/category/`<id>`/edit/	Редактирование	Изменение категории
* /shop/category/`<id>`/delete/	Удаление	Подтверждение удаления
* /shop/popular/	Популярные	Топ товаров по заказам
* /shop/recent-orders/	Заказы	Пользователи с заказами за 30 дней
* /shop/analytics/	Аналитика	Статистика и агрегаты
* /shop/ajax/product-catalog/	API	AJAX endpoint для каталога
* /admin/	Админка	Управление данными
