## 🛒 Интернет-магазин - Полный цикл: CRUD, корзина, авторизация

Учебный проект интернет-магазина на Django с реализацией:

- Расширенного CRUD для товаров и категорий
- Фильтрации, сортировки, пагинации (AJAX)
- Регистрации и авторизации пользователей
- Корзины покупок
- Аналитики и агрегатных запросов

---

## 📋 Функционал

### Основные возможности (для всех пользователей):

- **Просмотр каталога товаров** с фильтрацией по:

  - Категории товара
  - Диапазону цен
  - Дате добавления
- **Сортировка товаров** по популярности, цене и дате
- **Пагинация** - 10 товаров на страницу
- **AJAX подгрузка** - фильтрация и сортировка без перезагрузки
- **Страница популярных товаров** (топ по заказам)

### Для авторизованных пользователей:

- ✅ **Корзина покупок**

  - Добавление товаров
  - Изменение количества
  - Удаление товаров
  - Автоматический подсчёт суммы и количества

### Для администраторов (staff):

- ✅ **Управление категориями** (CRUD)
- ✅ **Управление товарами** (CRUD)
- ✅ **Загрузка изображений** для товаров
- ✅ **Страница недавних заказов** (статистика по пользователям)
- ✅ **Аналитика**:

  - Общая стоимость всех товаров
  - Количество товаров в каждой категории
  - Средняя, минимальная и максимальная цена по категориям
  - Общая статистика каталога

### Система авторизации:

- 🔐 Регистрация новых пользователей
- 🔐 Вход/выход из аккаунта
- 🔐 Разграничение прав (админ/пользователь)
- 🔐 Защита CSRF на всех формах

## 🗄️ Схема моделей

```
### Category (Категория)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| name | CharField(100) | Название категории (уникальное) |
| description | TextField | Описание категории |
| created_at | DateTimeField | Дата создания (авто) |

### Product (Товар)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| name | CharField(100) | Название товара |
| category | ForeignKey(Category) | Категория (связь) |
| price | DecimalField | Цена (валидация ≥ 0) |
| description | TextField | Описание товара |
| image | ImageField | Изображение товара |
| date_added | DateTimeField | Дата добавления (авто) |

### Cart (Корзина)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| user | OneToOneField(User) | Владелец корзины |
| created_at | DateTimeField | Дата создания |
| updated_at | DateTimeField | Дата обновления |

### CartItem (Товар в корзине)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| cart | ForeignKey(Cart) | Корзина |
| product | ForeignKey(Product) | Товар |
| quantity | PositiveIntegerField | Количество |

### Order (Заказ)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| user | ForeignKey(User) | Пользователь |
| created_at | DateTimeField | Дата создания |

### OrderItem (Товар в заказе)
| Поле | Тип | Описание |
|------|-----|----------|
| id | AutoField | Первичный ключ |
| order | ForeignKey(Order) | Заказ |
| product | ForeignKey(Product) | Товар |
| quantity | IntegerField | Количество |
```
